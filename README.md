<!-- ====================== FEATURED PROJECT ====================== -->
<h2 align="center">&#127769; Featured &#183; QALBI</h2>
<div align="center">
  QALBI is an AI mental wellness app I built for Malaysian youth. It runs CBT-style chat with self-harm
  detection, mood tracking, the DASS-21 screening, and a set of guided calming exercises. Everything is
  encrypted and private by design, so the people running it can't read your data. It started as my final
  year project and turned into the thing I work on now.
